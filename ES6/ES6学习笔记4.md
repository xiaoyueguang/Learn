# 正则的扩展

## RegExp 构造函数
`ES6`的`RegExp`能接受正则表达式作为参数
```
var regex = new RegExp('xyz', 'i');
//  等价于
var regex = /xyz/i;
//  等价于ES6
var regex = new RegExp(/xyz/i);
```
